<task type="auto">
  <name>Task 1: Scaffold Astro project with Tailwind v4</name>
  <files>package.json, astro.config.mjs, tsconfig.json, src/styles/global.css, src/config.ts</files>
  <action>
1. Initialize Astro project: `npm create astro@latest . -- --template minimal --no-install` (use minimal template, current directory).
2. Install dependencies: `npm install tailwindcss @tailwindcss/vite`.
3. Configure `astro.config.mjs`:
   - Import `defineConfig` and `fontProviders` from `"astro/config"`
   - Import `tailwindcss` from `"@tailwindcss/vite"`
   - Set `site: "https://fivexstrong.decoherance-interactive.com"`
   - Do NOT set `base` (defaults to `/` for CNAME subdomain)
   - Configure `fonts` array with Space Grotesk (`--font-space-grotesk`, `fontProviders.google()`) and Inter (`--font-inter`, `fontProviders.google()`)
   - Add `vite: { plugins: [tailwindcss()] }`
4. Create `src/styles/global.css` with the complete design token system:
   - `@import "tailwindcss";`
   - `@custom-variant dark (&:where(.dark, .dark *));` for dark-only site
   - `@theme {}` block with brand colors: `--color-brand-green: #00E676`, `--color-brand-teal: #00BFA5`, `--color-brand-purple: #D1C4E9`
   - Dark surface colors: `--color-surface-base: #0a0a0a`, `--color-surface-card: #141414`, `--color-surface-raised: #1a1a1a`
   - Border colors: `--color-border-subtle: #222222`, `--color-border: #2e2e2e`
   - Text colors: `--color-text-primary: #f5f5f5`, `--color-text-secondary: #b0b0b0`, `--color-text-muted: #717171`
   - Font-sans override: `--font-sans: var(--font-inter);`
   - `@theme inline {}` block wiring Fonts API variables: `--font-heading: var(--font-space-grotesk)`, `--font-body: var(--font-inter)`
   - `@layer base {}` with html styles: `background-color: var(--color-surface-base)`, `color: var(--color-text-primary)`, `font-family: var(--font-body)`
5. Create `src/config.ts` with exported constants:
   - `siteUrl: "https://fivexstrong.decoherance-interactive.com"`
   - `playStoreUrl: "https://play.google.com/store/apps/details?id=PLACEHOLDER_APP_ID"` (placeholder)
   - `ogTitle: "FiveX Strong — Strength training, simplified"`
   - `ogDescription: "The simplest way to run a 5x5 strength programme. Just pick a workout and lift."` (draft — Claude refine if needed)

NOTE: If Astro Fonts API requires `experimental: { fonts: true }` in config, add it. Try without first per research guidance.
  </action>
  <verify>Run `npm run build` — must complete with no errors and produce `dist/` directory. Check `dist/index.html` exists.</verify>
  <done>Astro project builds successfully with Tailwind v4, design tokens in global.css, fonts configured, and config.ts with site metadata.</done>
</task>
