<task type="auto">
  <name>Task 2: Create GitHub Actions workflow and static assets</name>
  <files>.github/workflows/deploy.yml, public/CNAME, public/og-image.png</files>
  <action>
1. Create `public/CNAME` with single line: `fivexstrong.decoherance-interactive.com` (no trailing newline or slash).

2. Create `.github/workflows/deploy.yml` (create `.github/workflows/` directories first):
   ```yaml
   name: Deploy to GitHub Pages
   on:
     push:
       branches: [main]
     workflow_dispatch:
   permissions:
     contents: read
     pages: write
     id-token: write
   concurrency:
     group: "pages"
     cancel-in-progress: false
   jobs:
     build:
       name: Build
       runs-on: ubuntu-latest
       steps:
         - name: Checkout
           uses: actions/checkout@v4
         - name: Setup Node
           uses: actions/setup-node@v4
           with:
             node-version: "22"
             cache: "npm"
             cache-dependency-path: package-lock.json
         - name: Install dependencies
           run: npm ci
         - name: Build with Astro
           run: npm run build
         - name: Upload artifact
           uses: actions/upload-pages-artifact@v3
           with:
             path: dist
     deploy:
       name: Deploy
       needs: build
       runs-on: ubuntu-latest
       environment:
         name: github-pages
         url: ${{ steps.deployment.outputs.page_url }}
       steps:
         - name: Deploy to GitHub Pages
           id: deployment
           uses: actions/deploy-pages@v5
   ```
   IMPORTANT: Do NOT use `actions/configure-pages` or inject `--base` flag. Custom subdomain means base is always "/".

3. Create placeholder OG image at `public/og-image.png` (1200x630):
   - Generate using an inline SVG converted to PNG. Use a Node script or ImageMagick if available:

     ```bash
     # Try with ImageMagick convert
     convert -size 1200x630 xc:'#0a0a0a' \

       -font Helvetica -pointsize 36 -fill '#f5f5f5' \
       -gravity center -annotate +0+40 'Strength training, simplified' \
       public/og-image.png
     ```
   - If ImageMagick not available, create a simple SVG and convert, or use a minimal canvas script.
   - Fallback: create a 1200x630 solid-color PNG with just the brand background color. The OG image is a placeholder to be swapped later — it just needs to be a valid PNG at the right dimensions.
   - Verify the file is a valid PNG and roughly 1200x630.
  </action>
