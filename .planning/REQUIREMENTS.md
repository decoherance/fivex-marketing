# Requirements: FiveX Strong Marketing Site

**Defined:** 2026-03-28
**Core Value:** Clearly communicate what FiveX Strong does and get visitors to download the app from Google Play.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Hero

- [ ] **HERO-01**: User sees app name "FiveX Strong" with brand-consistent headline
- [ ] **HERO-02**: User sees a 1-2 sentence value proposition explaining what the app does
- [ ] **HERO-03**: User can tap a prominent Google Play download button (official badge)
- [ ] **HERO-04**: User sees a placeholder phone mockup showing the app UI

### Features

- [ ] **FEAT-01**: User sees pre-defined workout programmes highlighted as a key benefit
- [ ] **FEAT-02**: User sees custom programme creation highlighted as a key benefit
- [ ] **FEAT-03**: User sees full history/progress tracking highlighted as a key benefit
- [ ] **FEAT-04**: User sees offline/no-internet capability highlighted as a key benefit

### Differentiators

- [ ] **DIFF-01**: User sees a prominent "no account needed" callout (not buried in features list)
- [ ] **DIFF-02**: User sees explicit "free forever, no paywall" messaging
- [ ] **DIFF-03**: User sees plain-language explanation that data stays on their device

### Screenshots

- [ ] **SCRN-01**: User sees app screenshot placeholder slots (2-4 screens)

### Final CTA

- [ ] **CTA-01**: User sees a closing section with Google Play download button

### Footer

- [ ] **FOOT-01**: Footer includes Google Play download link
- [ ] **FOOT-02**: Footer includes support contact (email)
- [ ] **FOOT-03**: Footer includes privacy policy link

### Technical

- [ ] **TECH-01**: Site is built with Astro generating static HTML
- [ ] **TECH-02**: Site is responsive and usable at 360px+ viewport
- [ ] **TECH-03**: Site uses dark theme with #00E676 green, Space Grotesk + Inter fonts
- [ ] **TECH-04**: Site deploys to GitHub Pages via GitHub Actions
- [ ] **TECH-05**: Site includes proper OG/meta tags for social sharing

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Content

- **CONT-01**: Real app screenshots replace placeholder slots
- **CONT-02**: Programme showcase with detailed descriptions of pre-defined programmes
- **CONT-03**: History/progress tracking visual demo

### Social Proof

- **SOCL-01**: Real download count or user stats when available
- **SOCL-02**: User testimonials or reviews from Play Store

### Analytics

- **ANLT-01**: Privacy-respecting analytics (Plausible or Fathom)

## Out of Scope

| Feature | Reason |
|---------|--------|
| Social proof / fake stats | App is new, no real numbers — fabricated stats destroy trust |
| Newsletter / email signup | Contradicts "no account needed" brand promise |
| Blog or content pages | Single page only, no maintenance burden |
| Video / trailer | No video content available |
| iOS App Store badge | App is Google Play only |
| Pop-ups or exit-intent modals | Interrupts conversion flow |
| Cookie consent banner | No third-party scripts or analytics in v1 |
| Pricing page | App is free, pricing section implies a catch |
| Contact form | Email link is sufficient |
| Community section | Not applicable for v1 |
